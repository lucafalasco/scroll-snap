Installation

Basic Usage

Here's a simple example of vertical snapping:

import createScrollSnap from 'scroll-snap'

const element = document.getElementById('container')
const { bind, unbind } = createScrollSnap(element, {
  snapDestinationY: '100%',
  timeout: 100,
  duration: 300,
  threshold: 0.2,
})
